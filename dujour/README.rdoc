== README

This is the web app called DuJour v0.2 built with Ruby 2.0.0

Dujour uses the Paperclip gem to manage photos, and Devise to manage authorizations

There is currently no seeds.db data, test information must be added manually through the UI.
Seed data will be added in future versions.

To start Dujour, just navigate to the proper file location in terminal and run 'rails s'
